В папке scan_dynamixel лежит проект для проверки коммутации
контроллера с двигателями. Если все правильно, в Serial monitor будут
выведены ID всех подключенных моторов. 
Note: моторы должны быть запитаны.

В папке demo лежит проект, который при исполнении ставит робота на ноги
и машет ручкой. Движение можно отключить закомментировав в loop-e hello.
